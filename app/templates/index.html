What is a database?

A database is a collection of structured data that is stored in a computer, which is managed by a Database Management System (DBS), together they are named as a database.

Nowadays there are many types of databases, but we can mention two very important types that are mostly used all over the world: relational databases and nonrelational or NoSQL databases.

Relational Databases

Relational databases are a type of database that use the relational model for structuring data into tables which are made of columns and rows, most of them use Structured Query Language (SQL).

You must first define the way you will store your data, this means you need to specify the tables and the structure of them before you can start storing data, which is a lot more restrictive.

SQL databases are good for multi-row transactions and are highly scalable.

These kinds of databases offer the most flexibility for accessing structured information.

Examples of SQL DBMS include MySQL, MariaDB, Oracle, PostgreSQL and MSSQL.

Nonrelational or NoSQL Databases

As the name implies, these type of databases don't use SQL to store and manipulate data, this means that the data stored is unstructured and/or semistructured. Data storage is a lot more flexible as it can use column-oriented, document-oriented, graph-based or Key-Value schemas to store it.

The types of queries you can make with NoSQL databases can't be too complicated unlike SQL databases.

Examples of NoSQL DBMS include MongoDB, Reddis, Cassandra, Elasticsearch and Firebase.
